=======
# crystal-frontend
CrystalVote frontend is a modern, responsive web application built with React, TypeScript, Vite, and Tailwind CSS. It delivers an intuitive user interface for secure elections, voting, and election management.

## ✨ Features

- Modern React architecture
- TypeScript for type safety
- Responsive UI with Tailwind CSS
- Secure authentication flow
- Election and voting interface
- Reusable component architecture
- Fast development with Vite

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- TanStack Query
- Axios
- Ant Design

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/crystal-frontend.git
```

Navigate into the project:

```bash
cd crystal-frontend
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
